# Cloudflare ドメイン取得手順

**ドメイン**: tokyora.com
**価格**: $9.77/年（約1,450円）
**所要時間**: 15分

---

## ステップ1: Cloudflareアカウント作成（5分）

### 1.1 サインアップ
1. https://dash.cloudflare.com/sign-up にアクセス
2. メールアドレス入力: `taka.suzuka@kokagaku.com` 推奨
3. パスワード設定（強力なものを）
4. 確認メールのリンクをクリック

### 1.2 ログイン
- https://dash.cloudflare.com にログイン

---

## ステップ2: ドメイン購入（10分）

### 2.1 Registrarにアクセス
1. ダッシュボード左メニュー → **"Domain Registration"** をクリック
2. または直接 https://dash.cloudflare.com/?to=/:account/domains にアクセス

### 2.2 ドメイン検索
1. 検索ボックスに **`tokyora.com`** を入力
2. "Search" をクリック
3. 「Available」と表示されることを確認

### 2.3 購入手続き
1. **"Purchase"** ボタンをクリック
2. **Auto-Renew**: ON（自動更新推奨）
3. **Contact Information** を入力:
   ```
   First Name: ten（または本名）
   Last Name: （本名）
   Email: taka.suzuka@kokagaku.com
   Phone: +81-90-XXXX-XXXX
   Address: （青山の住所）
   City: Minato-ku
   State/Province: Tokyo
   Postal Code: （郵便番号）
   Country: Japan
   ```

4. **Whois Privacy**: 自動的に有効（無料）

### 2.4 支払い
1. **Payment Method**: クレジットカード追加
2. カード情報入力
3. **"Complete Purchase"** をクリック

---

## ステップ3: DNS設定（後で）

**重要**: ドメイン購入直後はDNS設定不要
- Vercelでサイト構築後に設定します
- その時に別途手順書を提供します

2. Cloudflareダッシュボードの "Domain Registration" に tokyora.com が表示される

## 🔒 セキュリティ設定（推奨）

### 2要素認証（2FA）を有効化
1. ダッシュボード右上 → プロフィールアイコン
2. "My Profile" → "Authentication"
3. "Two-Factor Authentication" → "Set up"
4. 認証アプリ（Google Authenticator等）でQRコード読み取り

## ❓ トラブルシューティング

### Q: ドメインが見つからない
A: 誰かが先に取得した可能性。すぐにミニちゃんに報告してください。

### Q: 支払いが通らない
A: クレジットカードの国際決済が有効か確認。または別のカードを試す。

### Q: 確認メールが届かない
A: 迷惑メールフォルダを確認。それでもなければ support@cloudflare.com に問い合わせ。
